## Features

IZI Link offers a wide range of features to help you build a complete payment solution:

- **Payment Processing**: Accept payments online, in-person, or via links
- **Customer Management**: Store and manage customer information securely
- **Invoicing**: Create and send professional invoices to your customers
- **Reporting**: Generate detailed reports on your payment activity
- **Webhooks**: Receive real-time notifications about events in your account
- **SDKs & Libraries**: Use our official client libraries for your preferred language

## Authentication

All API requests to IZI Link require authentication. We use API keys to authenticate requests. You can view and manage your API keys in the [IZI Link Dashboard](https://app.izilink.com).

```bash
# Example API request with authentication
curl -X GET \
  https://api.izilink.com/v1/payments \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Learn more about [authentication](/authentication) in our documentation.
