## Eliminar uma Referência

Cancela uma referência válida, tornando-a indisponível para pagamento.

### Autenticação

<ParamField header="Authorization" type="string" required>
    Assinatura HMAC no formato: `ApiKey AppId:hash:nonce:timestamp`
</ParamField>

<ParamField query="AppId" type="string" required>
    O seu ID de aplicação fornecido pelo IZI Pay
</ParamField>

<ParamField query="AppSecret" type="string" required>
    O seu segredo de aplicação fornecido pelo IZI Pay (codificado em Base64)
</ParamField>

### Parâmetros de Caminho

<ParamField path="id" type="string" required>
    ID único da referência
</ParamField>

### Exemplo de Requisição

```bash
curl --request DELETE \
  --url https://api-sandbox.izipay.ao/v1/referencias/00000000-0000-0000-0000-000000000000 \
  --header 'Authorization: ApiKey YOUR_HMAC_SIGNATURE'
```

### Resposta

Nenhum conteúdo é retornado para uma requisição bem-sucedida.

### Códigos de Resposta

| Código de Estado | Descrição                                         |
|------------------|---------------------------------------------------|
| 204              | Sem Conteúdo - Referência eliminada com sucesso   |
| 400              | Pedido Inválido - Requisição inválida             |
| 401              | Não Autorizado - Assinatura inválida              |
| 402              | Pagamento Necessário - Conta em dívida            |
| 422              | Entidade Não Processável - Referência inexistente |
