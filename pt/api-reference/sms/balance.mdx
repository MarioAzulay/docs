# Saldo de SMS

Monitorize o saldo de créditos de SMS e veja o histórico de transações para gerir eficazmente os seus recursos de mensagens.

## URL Base

Todas as requisições à API devem ser feitas para a seguinte URL base:

```
https://sms.izipay.ao
```

## Obter Saldo Atual

Recupere o saldo atual de créditos de SMS.

<ParamField path="Saldo/ObterSaldo" type="string">
    Endpoint para obter o saldo atual de SMS.
</ParamField>

### Cabeçalhos

<ParamField header="Authorization" type="string" required>
    Token Bearer recebido do endpoint de autenticação. Formato: `Bearer SEU_TOKEN`
</ParamField>

### Exemplo de Requisição

<CodeGroup>

    ```bash cURL
    curl -X GET \
    https://sms.izipay.ao/api/Saldo/ObterSaldo \
    -H "Authorization: Bearer SEU_TOKEN"
    ```

    ```javascript Node.js
    const axios = require('axios');

    const response = await axios.get('https://sms.izipay.ao/api/Saldo/ObterSaldo', {
    headers: {
    'Authorization': 'Bearer SEU_TOKEN'
}
});

    console.log('Saldo de SMS:', response.data);
    ```

    ```python Python
    import requests

    response = requests.get(
    'https://sms.izipay.ao/api/Saldo/ObterSaldo',
    headers={
    'Authorization': 'Bearer SEU_TOKEN'
}
    )

    data = response.json()
    print(f"Saldo de SMS: {data}")
    ```

</CodeGroup>

## Formato de Resposta

### Resposta de Saldo

<ResponseField name="saldo" type="number">
    Saldo atual de créditos de SMS.
</ResponseField>

<ResponseField name="ultimaAtualizacao" type="string">
    Data e hora da última atualização do saldo.
</ResponseField>

### Resposta de Histórico de Saldo

<ResponseField name="items" type="array">
    Lista de entradas do histórico de saldo.
</ResponseField>

<ResponseField name="items[].id" type="integer">
    Identificador único da entrada do histórico.
</ResponseField>

<ResponseField name="items[].data" type="string">
    Data e hora da alteração do saldo.
</ResponseField>

<ResponseField name="items[].tipo" type="string">
    Tipo de transação (ex.: "Crédito", "Débito").
</ResponseField>

<ResponseField name="items[].valor" type="number">
    Quantidade de créditos adicionados ou utilizados.
</ResponseField>

<ResponseField name="items[].descricao" type="string">
    Descrição da transação.
</ResponseField>

## Códigos de Erro

| Código | Descrição                                   |
|--------|---------------------------------------------|
| 401    | Não autorizado - Token inválido ou expirado |
| 403    | Proibido - Permissões insuficientes         |
| 500    | Erro no servidor                            |

## Melhores Práticas

- Monitorize regularmente o seu saldo de SMS para evitar interrupções
- Configure alertas para limites baixos de saldo
- Analise o histórico do saldo para acompanhar padrões de uso
- Mantenha as informações da empresa sempre atualizadas
- Implemente controles de acesso adequados para gestão de saldo
