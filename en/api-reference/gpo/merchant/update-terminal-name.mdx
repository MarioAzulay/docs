## Update Terminal Name

This service allows you to update the name of a terminal.

### Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token in the format: `Bearer {token}`
</ParamField>

### Path Parameters

<ParamField path="posId" type="string" required>
  Terminal ID
</ParamField>

### Request Body

<ParamField body="name" type="string" required>
  New terminal name
</ParamField>

### Response

<ResponseField name="result" type="object">
  <Expandable title="Properties">
    <ResponseField name="id" type="string">
      Terminal ID
    </ResponseField>
    <ResponseField name="name" type="string">
      Updated terminal name
    </ResponseField>
    <ResponseField name="state" type="string">
      Terminal state
    </ResponseField>
    <ResponseField name="createdDate" type="string">
      Creation date
    </ResponseField>
    <ResponseField name="updatedDate" type="string">
      Update date
    </ResponseField>
    <ResponseField name="establishment" type="object">
      <Expandable title="Properties">
        <ResponseField name="id" type="string">
          Establishment ID
        </ResponseField>
        <ResponseField name="name" type="string">
          Establishment name
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="merchant" type="object">
      <Expandable title="Properties">
        <ResponseField name="id" type="string">
          Merchant ID
        </ResponseField>
        <ResponseField name="name" type="string">
          Merchant name
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl --request PUT \
  --url "https://gpo-sandbox.izipay.ao/v1/points-of-sale/413206" \
  --header "Authorization: Bearer YOUR_TOKEN" \
  --header "Content-Type: application/json" \
  --data '{
    "name": "New Terminal Name"
  }'
```
