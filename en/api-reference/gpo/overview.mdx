# Gateway de Pagamento Online (GPO)

The Gateway de Pagamento Online (GPO) system provides APIs for various functionalities, including authorizations, payments, refunds, terminal management, and user management.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token in the format: `Bearer {token}`
</ParamField>

To obtain an access token, use the following endpoint:

```bash
POST {{baseuri}}/v2/AccessToken
```

Request body:
```json
{
  "username": "your_username",
  "password": "your_password"
}
```
