## Get WebFrame Details

This service allows you to retrieve detailed information about a specific WebFrame by its ID.

### Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token in the format: `Bearer {token}`
</ParamField>

### Path Parameters

<ParamField path="merchantId" type="string" required>
  Merchant ID
</ParamField>

<ParamField path="webframeId" type="string" required>
  WebFrame ID
</ParamField>

### Response

<ResponseField name="id" type="string">
  WebFrame ID
</ResponseField>

<ResponseField name="url" type="string">
  URL to access the WebFrame
</ResponseField>

<ResponseField name="amount" type="number">
  Payment amount
</ResponseField>

<ResponseField name="currency" type="string">
  Currency code
</ResponseField>

<ResponseField name="expirationDate" type="string">
  Expiration date in ISO 8601 format
</ResponseField>

<ResponseField name="merchantReferenceNumber" type="string">
  Merchant reference number
</ResponseField>

<ResponseField name="description" type="string">
  Description of the payment
</ResponseField>

<ResponseField name="callbackUrl" type="string">
  URL to be called after payment completion
</ResponseField>

<ResponseField name="successUrl" type="string">
  URL to redirect the customer after successful payment
</ResponseField>

<ResponseField name="cancelUrl" type="string">
  URL to redirect the customer if payment is cancelled
</ResponseField>

<ResponseField name="status" type="string">
  WebFrame status (ACTIVE, EXPIRED, COMPLETED, CANCELLED)
</ResponseField>

<ResponseField name="createdDate" type="string">
  Creation date
</ResponseField>

<ResponseField name="updatedDate" type="string">
  Update date
</ResponseField>

<ResponseField name="merchant" type="object">
  <Expandable title="Properties">
    <ResponseField name="id" type="string">
      Merchant ID
    </ResponseField>
    <ResponseField name="name" type="string">
      Merchant name
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="payment" type="object">
  <Expandable title="Properties">
    <ResponseField name="id" type="string">
      Payment ID
    </ResponseField>
    <ResponseField name="transactionType" type="string">
      Transaction type
    </ResponseField>
    <ResponseField name="transactionState" type="string">
      Transaction state
    </ResponseField>
    <ResponseField name="amount" type="number">
      Payment amount
    </ResponseField>
    <ResponseField name="currency" type="string">
      Currency code
    </ResponseField>
    <ResponseField name="createdDate" type="string">
      Creation date
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl --request GET \
  --url "https://gpo-sandbox.izipay.ao/v1/merchants/123456/webframes/WF8TU5PDFXH76GVLT8" \
  --header "Authorization: Bearer YOUR_TOKEN"
```

### Response Codes

| Status Code | Description |
|-------------|-------------|
| 200 | OK - WebFrame details retrieved successfully |
| 401 | Unauthorized - Invalid token |
| 404 | Not Found - Merchant or WebFrame not found |
