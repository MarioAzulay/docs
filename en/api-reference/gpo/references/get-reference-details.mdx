## Get Reference Details

This service allows you to retrieve detailed information about a specific payment reference by its ID.

### Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token in the format: `Bearer {token}`
</ParamField>

### Path Parameters

<ParamField path="merchantId" type="string" required>
  Merchant ID
</ParamField>

<ParamField path="referenceId" type="string" required>
  Reference ID
</ParamField>

### Response

<ResponseField name="id" type="string">
  Reference ID
</ResponseField>

<ResponseField name="amount" type="number">
  Payment amount
</ResponseField>

<ResponseField name="currency" type="string">
  Currency code
</ResponseField>

<ResponseField name="expirationDate" type="string">
  Expiration date in ISO 8601 format
</ResponseField>

<ResponseField name="merchantReferenceNumber" type="string">
  Merchant reference number
</ResponseField>

<ResponseField name="description" type="string">
  Description of the payment reference
</ResponseField>

<ResponseField name="createdDate" type="string">
  Creation date
</ResponseField>

<ResponseField name="updatedDate" type="string">
  Update date
</ResponseField>

<ResponseField name="merchant" type="object">
  <Expandable title="Properties">
    <ResponseField name="id" type="string">
      Merchant ID
    </ResponseField>
    <ResponseField name="name" type="string">
      Merchant name
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl --request GET \
  --url "https://gpo-sandbox.izipay.ao/v1/merchants/123456/references/REF8TU5PDFXH76GVLT8" \
  --header "Authorization: Bearer YOUR_TOKEN"
```

### Example Response

```json
{
  "id": "REF8TU5PDFXH76GVLT8",
  "amount": 1000.00,
  "currency": "AOA",
  "expirationDate": "2023-12-31T23:59:59Z",
  "merchantReferenceNumber": "REF123456",
  "description": "Payment for order #123456",
  "createdDate": "2023-01-01T12:00:00Z",
  "updatedDate": "2023-01-01T12:00:00Z",
  "merchant": {
    "id": "123456",
    "name": "Merchant Name"
  }
}
```

### Response Codes

| Status Code | Description |
|-------------|-------------|
| 200 | OK - Reference details retrieved successfully |
| 401 | Unauthorized - Invalid token |
| 404 | Not Found - Merchant or reference not found |
