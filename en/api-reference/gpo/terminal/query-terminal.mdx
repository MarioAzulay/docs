## Query Terminal

This service allows you to query information about a terminal. The **PointOfSale** resource is returned by the service with information regarding the terminal.

### Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token in the format: `Bearer {token}`
</ParamField>

### Path Parameters

<ParamField path="posId" type="string" required>
  Terminal ID
</ParamField>

### Response

<ResponseField name="result" type="object">
  <Expandable title="Properties">
    <ResponseField name="id" type="string">
      Terminal ID
    </ResponseField>
    <ResponseField name="name" type="string">
      Terminal name
    </ResponseField>
    <ResponseField name="state" type="string">
      Terminal state (OPEN, CLOSED)
    </ResponseField>
    <ResponseField name="createdDate" type="string">
      Creation date
    </ResponseField>
    <ResponseField name="updatedDate" type="string">
      Update date
    </ResponseField>
    <ResponseField name="establishment" type="object">
      <Expandable title="Properties">
        <ResponseField name="id" type="string">
          Establishment ID
        </ResponseField>
        <ResponseField name="name" type="string">
          Establishment name
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="merchant" type="object">
      <Expandable title="Properties">
        <ResponseField name="id" type="string">
          Merchant ID
        </ResponseField>
        <ResponseField name="name" type="string">
          Merchant name
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="currentClearingPeriod" type="object">
      <Expandable title="Properties">
        <ResponseField name="number" type="integer">
          Accounting period number
        </ResponseField>
        <ResponseField name="openDate" type="string">
          Opening date
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl --request GET \
  --url "https://gpo-sandbox.izipay.ao/v1/points-of-sale/413206" \
  --header "Authorization: Bearer YOUR_TOKEN"
```

### Response Codes

| Status Code | Description |
|-------------|-------------|
| 200 | OK - Terminal information retrieved successfully |
| 401 | Unauthorized - Invalid token |
| 404 | Not Found - Terminal not found |
