# SMS Templates

Create and manage reusable SMS templates to streamline your messaging campaigns.

## Base URL

All API requests should be made to the following base URL:

```
https://sms.izipay.ao
```

## Authentication

All requests require an `Authorization` header with a Bearer token:

```
Authorization: Bearer YOUR_TOKEN
```

## List SMS Templates

Retrieve all SMS templates associated with your account.

### Example Request

<CodeGroup>

    ```bash cURL
    curl -X GET \
    "https://sms.izipay.ao/api/TemplatesSMS/ListarTemplatesSms" \
    -H "Authorization: Bearer YOUR_TOKEN"
    ```

    ```javascript Node.js
    const axios = require('axios');

    async function fetchTemplates() {
    const response = await axios.get(
    'https://sms.izipay.ao/api/TemplatesSMS/ListarTemplatesSms',
{headers: {'Authorization': 'Bearer YOUR_TOKEN'}}
    );
    console.log('SMS Templates:', response.data);
}
    ```

    ```python Python
    import requests

    def fetch_templates():
    response = requests.get(
    'https://sms.izipay.ao/api/TemplatesSMS/ListarTemplatesSms',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
    )
    print("SMS Templates:", response.json())
    ```

</CodeGroup>

## Send SMS Using Template

Send an SMS using a predefined template.

### Example Request

<CodeGroup>

    ```bash cURL
    curl -X POST \
    "https://sms.izipay.ao/api/Envio/enviarSmsTemplate" \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer YOUR_TOKEN" \
    -d '{
    "smsId": 123,
    "numerosDestino": ["244999999999", "244888888888"],
    "remetente": "IZI PAY"
}'
    ```

    ```javascript Node.js
    async function sendSms(smsId, numerosDestino, remetente) {
    const response = await axios.post(
    'https://sms.izipay.ao/api/Envio/enviarSmsTemplate',
{smsId, numerosDestino, remetente},
{headers: {'Authorization': 'Bearer YOUR_TOKEN', 'Content-Type': 'application/json'}}
    );
    console.log('SMS sent:', response.data);
}
    ```

    ```python Python
    def send_sms(sms_id, numeros_destino, remetente):
    response = requests.post(
    'https://sms.izipay.ao/api/Envio/enviarSmsTemplate',
    json={"smsId": sms_id, "numerosDestino": numeros_destino, "remetente": remetente},
    headers={'Authorization': 'Bearer YOUR_TOKEN', 'Content-Type': 'application/json'}
    )
    print("SMS sent:", response.json())
    ```

</CodeGroup>

## Response Format

<ResponseField name="success" type="boolean">
    Indicates whether the operation was successful.
</ResponseField>

<ResponseField name="data" type="object">
    Contains the template data or operation result.
</ResponseField>

<ResponseField name="message" type="string">
    A message describing the result of the operation.
</ResponseField>

## Error Codes

| Code | Description                                       |
|------|---------------------------------------------------|
| 400  | Invalid request format or missing required fields |
| 401  | Unauthorized - Invalid or expired token           |
| 403  | Forbidden - Insufficient permissions              |
| 404  | Template not found                                |
| 500  | Server error                                      |

## Best Practices

- Use reusable templates to save time and ensure consistency.
- Validate phone numbers before sending SMS messages.
- Implement error logging to troubleshoot issues effectively.
- Test templates before using them in production.
- Keep messages concise to avoid extra costs and ensure clarity.
- Regularly update templates to reflect any necessary changes.
